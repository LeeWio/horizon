- **JPA/Hibernate** - ORM映射
- **MySQL** - 主数据库

- **UUID主键** - 分布式友好

### **安全层**
- **Spring Security** - 安全框架
- **JWT认证** - 无状态认证

## 📝 **使用说明**

### **启动项目**
```bash
# 编译项目
mvn clean compile

# 启动应用
mvn spring-boot:run

# 访问API文档
http://localhost:8080/swagger-ui.html
```

### **数据库初始化**
```bash
# 执行初始化脚本
mysql -u root -p horizon_blog < src/main/resources/sql/init.sql
```
